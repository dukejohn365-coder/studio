# **App Name**: FocusFlow

## Core Features:

- Task Creation: Allow users to create and add new tasks to their to-do list.
- Task Categorization: Enable users to categorize tasks using tags for better organization.
- Task Prioritization: Allow users to set priorities for tasks, influencing their display order.
- Deadline Setting: Implement functionality for users to set deadlines for each task.
- Progress Tracking: Enable users to track the completion status of each task with visual progress indicators.
- Landing Page: Develop a marketing landing page to showcase the app's features and benefits.

## Style Guidelines:

- Primary color: Calm blue (#73A580) to inspire focus and tranquility.
- Background color: Soft beige (#F5F5DC) to reduce eye strain during long work sessions.
- Accent color: Muted orange (#D2691E) to highlight important actions and priorities.
- Body and headline font: 'PT Sans' (humanist sans-serif) for a modern, yet approachable look, ensuring readability for prolonged use.
- Use minimalist line icons to represent tasks and categories, ensuring clarity and focus.
- Employ a clean, card-based layout to present tasks in an organized and visually appealing manner.
- Incorporate subtle animations for task transitions and feedback, enhancing the user experience without distraction.
